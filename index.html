I'm currently employed at Phoenix Contact

During my tenure at Phoenix, I've worked on a web-based configurator for a test disconnect system. It is currently in use for internal customization, allowing module selection with size customization, marker color and text customization, test pole color and position, and test plug type selection; with bill of material output into pdf and importable excel data. Another project I worked on was a configurator for DIN Rail, which was made with a responsive Bootstrap 3 front end and a PHP / mySQL backend. This configurator stored login information in tandem with DIN Rail information, which in addition to having a dynamic DIN Rail selection, allowed for new DIN Rail part numbers to be added in the DB and viewed instantly! In my time on this planet, I've worked with HTML, CSS, Bootstrap 3 & 4, Bulma, Javascript, JQuery, PHP, MySql, Apache, Python, and Papyrus
