I'm currently employed at Elite Property Management as an IT Systems Admin

Currently, I'm working on an Inventory Management System, with a Bootstrap 4 front-end and PHP backend. It utilizes prepared statements, for protection from SQL injection. As well as Periscope data visualization dashboard, which allow for increased productivity for our team.

During my previous tenure at Phoenix, I'd worked on a web-based configurator for a test disconnect system. It was in use for internal customization, allowing module selection with size customization, marker color and text customization, test pole color and position, and test plug type selection; with bill of material output into pdf and importable excel data. Another project I'd worked on was a configurator for DIN Rail, which was made with a responsive Bootstrap 3 front end and a PHP / mySQL backend. This configurator stored login information in tandem with DIN Rail information, which in addition to having a dynamic DIN Rail selection, allowed for new DIN Rail part numbers to be added in the DB and viewed instantly! In my time on this planet, I've worked with HTML, CSS, Bootstrap 3 & 4, Bulma, Javascript, JQuery, PHP, MySql, Apache, Python, and Papyrus
