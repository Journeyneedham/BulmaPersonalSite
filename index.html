I'm currently employed at Elite Property Management as an IT Systems Admin

Currently, I'm currently working on an application for filtering applicants during the initial stage, which inserts applicant info and scrapes the results of various public sites. During my previous tenure at Phoenix Contact, I'd worked on a web-based configurator for a test disconnect system. This configurator had user login, and server-side configuration storage!

During my time in development, I've worked with HTML, CSS, Bootstrap 3 & 4, Bulma, Javascript (JQuery), PHP, MySQL & PostgreSQL, Apache / AWS / Heroku, Python (Flask & Django), and Papyrus
